Definição de Web

Web é uma palavra inglesa que significa teia ou rede. Com o surgimento da Internet, a web passou a designar a rede que conecta computadores por todo mundo, a Wold Wide Web (www).

A web significa um sistema de informações ligadas através de hipermídia (hiperligações em forma de texto, vídeo, som e outras animações digitais) que permitem ao usuário acessar uma infinidade de contéudos através da Internet. Para tal é necessário conexão com a Internet e um navegador (browser) onde são visualizados os contéudos disponíveis.

Website

No contexto das comunicaçãos eletrônicas, website e site possuem o mesmo significado e são utilizadas para fazer referência a uma página ou a uma agrupamento de páginas relacionadas entre si, acessíveis na Internet através de um determinado endereço.

Cada website possui um objetivo, de acordo com o público ao qual é direcionado. As páginas web são desenvolvidas com linguagens de marcação como o HTML, as quais podem ser interpretadas pelos navegadores. Nas páginas de um website podem ser colocados textos, imagens, vídeos ou animações digitais. Estas páginas são carregas através do protocolo de rede HTTP (Hypertext Transfer Protocol) e são visualidas através de um navegador (brower).

Web é o nome dado aos sistemas de contéudo disponíveis na Internet. E é ao ato de clicar em links, visitar sites e interagir com o contéudo oferecido na web que se chama navegador. Um website pode apresentar uma forma estática, utilizando a linguagem HTML, ou uma forma dinâmica, interativa, utilizando linguagens como JavaScript e PHP.

Portanto, uma página web é o nome que se dá a um documento eletrônico. As páginas web geralmente se encontram em formato HTML, que é a linguagem utilizada nelas e que permite incluir diversos elementos, como os links, por exemplo.

Conceitos relacionados:

URL
Uniform Resource Locator - Específica o endereço único que cada página vai receber. É o texto que se digita na barra de endereços de seu navegador para acessar uma determinada página ou serviço.
HTTP
Hypertext Transfer Protocol - Protocolo de Comunicação que permite a transferência de informação entre redes.
HTML
HyperText Markup Language - Permite a construção de websites e a inserção de novos contéudos, como imagens e vídeos, por meio dos hipertextos.

Breve História da Internet

A Internet nasceu como uma rede fechada, nos anos 60, com o nome de Aparnet. Criada em laboratórios militares dos Estados Unidos, servia para trocar informações entre computadores do governo. Só em 1989 a proposta ganhou a características que conhecemos hoje com o surgimento da www (Wold Wide Web).

Desenvolvidas pelo físico inglês Tim Berners-Lee, nos laboratórios da Organização Europeia para a Pesquisa Nuclear (CERN), a www estabeleceu uma linguagem padrão para a circulação de dados na rede, permitindo que qualquer computador, de qualquer parte do planeta, tivesse livre acesso ao mundo virtual.

A partir da décade de 90 ocorreu o "boom da internet", pois foi quando ela se popularizou pelo mundo, com o surgimento de novos browers e o aumento do número de usuários.
